第1条　はじめに

「乗り鉄GO」（以下「本アプリ」）は、鉄道駅・路線の訪問記録、写真投稿、路線制覇率の管理を楽しむためのアプリケーションです。本アプリを運営する＿＿＿＿（以下「当方」）は、ユーザーの個人情報を適切に保護することを重要な責務と考え、以下のとおりプライバシーポリシー（以下「本ポリシー」）を定めます。

本アプリをご利用になる前に、本ポリシーをよくお読みください。本アプリをご利用いただくことで、本ポリシーの内容をご確認いただいたものとして取り扱います。本ポリシーの内容に同意いただけない場合は、本アプリのご利用をお控えください。

第2条　収集する情報

本アプリは、サービス提供のために以下の情報を収集・保存します。

2-1. アカウント情報

  • Firebase Authentication が発行するユーザーID（UID）
  • 認証プロバイダ情報（Sign in with Apple を使用する場合のApple IDに関する認証情報）
  • メールアドレス（Sign in with Apple を利用する場合、ユーザーの設定によってはAppleのリレーアドレスが取得されます。メールアドレスが取得されない場合もあります）
  • アカウント作成日時・最終更新日時

2-2. プロフィール情報

  • 表示名（ユーザーが任意で設定・変更するもの）

2-3. 利用データ

  • 訪問した駅のID・駅名・路線名
  • 訪問日時
  • 訪問メモ（任意入力）
  • 訪問回数・初回訪問日・最終訪問日

2-4. 投稿・コンテンツデータ

  • ユーザーが投稿した写真（Firebase Storage に保存）
  • 投稿に紐づく駅名・路線名・投稿種別・メモ（任意）
  • 投稿の作成日時・ステータス

2-5. ソーシャルデータ

  • いいね情報（対象投稿ID・いいねしたユーザーID・日時）
  • コメント本文・投稿日時・コメントしたユーザーID・表示名

2-6. 通報データ

  • 通報対象の投稿ID・対象ユーザーID・通報理由・通報者UID・通報日時

通報されたコンテンツは、不正利用防止・コミュニティの安全確保を目的として当方が確認・審査を行う場合があります。審査のため、通報に関連するコンテンツ（投稿・コメント等）を参照・保存することがあります。

2-7. Firebaseが自動収集する情報

本アプリはFirebase（Google LLC が提供するサービス）を利用しており、サービス提供・セキュリティ確保・不正利用防止の目的で、IPアドレス、デバイス識別子、ネットワーク状態などの情報がFirebaseによって自動的に収集される場合があります。これらの情報の収集・利用はGoogle LLCのプライバシーポリシーに従います。

第3条　収集しない情報

本アプリは、以下の情報を収集しません。

  • 位置情報・GPS情報
  • 連絡先・電話帳情報
  • 決済情報・クレジットカード番号
  • ヘルスケア・医療情報
  • 広告識別子（IDFA等）
  • マイクへのアクセス（本アプリはマイクを使用しません）
  • カメラの直接起動（写真投稿時はフォトライブラリからの選択のみ行います。カメラの直接起動は行いません）

第4条　情報の利用目的

収集した情報は、以下の目的のためにのみ利用します。

  1. 本アプリのサービス提供（訪問記録の管理、投稿機能、いいね・コメント機能など）
  2. ユーザーアカウントの識別・認証
  3. 不正利用の検出・防止およびサービスの安全性確保
  4. 通報に基づくコンテンツの審査・対応およびコミュニティの健全性維持
  5. サービスの改善・品質向上
  6. 法令上の義務の履行

第5条　第三者提供

当方は、以下の場合を除き、収集した情報を第三者に提供しません。

  1. ユーザー本人の同意がある場合
  2. 法令に基づく場合（捜査機関からの適法な要請など）
  3. 人の生命・身体・財産の保護のために必要であり、本人の同意を得ることが困難な場合

委託先（サービスプロバイダ）

本アプリは、以下のサービスを利用しており、これらのサービスに必要な範囲でデータが処理されます。

サービス 提供元 用途
Firebase Authentication Google LLC ユーザー認証
Cloud Firestore Google LLC データベース（訪問記録・投稿・コメント等）
Firebase Storage Google LLC 写真ファイルの保存

Google LLCのプライバシーポリシーは https://policies.google.com/privacy をご参照ください。

第6条　データの保存期間と削除

アカウント削除時

ユーザーが本アプリの設定画面からアカウント削除を実行した場合、以下のデータの削除処理が行われます。

  • ユーザーアカウント情報（表示名、作成日時等）
  • 訪問記録（訪問した駅の記録、訪問日時、メモ）
  • 自分が投稿した写真および投稿データ
  • 認証アカウント（Firebase Authentication）
システム上の処理タイミングや技術的な制約により、他のユーザーの投稿に紐づくいいね・コメントなど一部のデータは、削除処理に時間差が生じる場合があります。また、不正利用の防止、法令上の義務の履行、サービスの保全等の目的のために必要と判断される場合（通報データ等）は、アカウント削除後も一定期間データを保持することがあります。

アカウント削除後

削除処理完了後は、同一アカウントでの再ログインはできません。

第7条　ユーザーの権利

ユーザーは以下の操作をアプリ内から行うことができます。

  1. アカウント削除：設定画面（マイページ → 設定 → 危険な操作 → アカウントを削除する）からいつでも実行できます
  2. 表示名の変更：マイページの編集機能から変更できます
  3. 投稿の削除：自分が投稿したコンテンツはアプリ内から削除できます

その他、収集したデータに関するお問い合わせは、下記お問い合わせ先までご連絡ください。

第8条　セキュリティ

当方は、収集した情報の漏洩・紛失・改ざんを防止するため、Firebaseが提供するセキュリティ機能（Firestore Security Rules、Firebase Storage Security Rules等）を活用し、適切な管理措置を講じます。ただし、インターネット上でのデータ送受信において完全なセキュリティを保証することはできません。

第9条　お子様のプライバシー

本アプリは13歳未満のお子様を対象としていません。13歳未満の方が本アプリを使用されていることが判明した場合、速やかにアカウントを削除し、関連データを消去します。

第10条　本ポリシーの変更

当方は、法令の変更やサービスの改善に伴い、本ポリシーを変更することがあります。重要な変更については、本アプリ内またはウェブサイト上でお知らせします。変更後も本アプリを継続してご利用になった場合、変更後のポリシーの内容をご確認いただいたものとして取り扱います。
